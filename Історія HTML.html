Історія мови гіпертекстової розмітки HTML

Вступ

HTML (HyperText Markup Language) є стандартною мовою розмітки для створення веб-сторінок. HTML описує структуру веб-сторінки за допомогою елементів, які браузери використовують для відображення контенту.

Історія HTML

Початок (1989-1991)

HTML був розроблений Тимом Бернерс-Лі в 1989 році, коли він працював у CERN. Його метою було створення системи для обміну документами між вченими по всьому світу. Перший документ HTML було написано в 1991 році, і він включав опис 18 елементів, з яких більшість використовуються й сьогодні.

HTML 2.0 (1995)

HTML 2.0 був офіційно прийнятий у 1995 році і став першим стандартом HTML, який описував усі базові можливості мови, на які могли покладатися браузери того часу.

HTML 3.2 та HTML 4.0 (1997)

HTML 3.2 був прийнятий у 1997 році, вводячи нові елементи для таблиць та математичних формул. Пізніше того ж року було прийнято HTML 4.0, який додав підтримку для CSS (Cascading Style Sheets) і значно розширив можливості мови.

XHTML (2000)

XHTML був спробою зробити HTML більш структурованим і XML-сумісним. XHTML 1.0 було прийнято у 2000 році.

HTML5 (2014)

HTML5 є сучасною версією HTML, яка включає нові елементи для мультимедіа, графіки, інтерактивних додатків та поліпшення семантичної структури документів. HTML5 став офіційним стандартом у 2014 році і постійно оновлюється для підтримки нових технологій.
