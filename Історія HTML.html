    <section>

        <h2>Історія HTML</h2>

        <h3>Початок (1989-1991)</h3>

        <p>HTML був розроблений Тимом Бернерс-Лі в 1989 році, коли він працював у CERN. Його метою було створення системи для обміну документами між вченими по всьому світу. Перший документ HTML було написано в 1991 році, і він включав опис 18 елементів, з яких більшість використовуються й сьогодні.</p>

        

        <h3>HTML 2.0 (1995)</h3>

        <p>HTML 2.0 був офіційно прийнятий у 1995 році і став першим стандартом HTML, який описував усі базові можливості мови, на які могли покладатися браузери того часу.</p>

        

        <h3>HTML 3.2 та HTML 4.0 (1997)</h3>

        <p>HTML 3.2 був прийнятий у 1997 році, вводячи нові елементи для таблиць та математичних формул. Пізніше того ж року було прийнято HTML 4.0, який додав підтримку для CSS (Cascading Style Sheets) і значно розширив можливості мови.</p>

        

        <h3>XHTML (2000)</h3>

        <p>XHTML був спробою зробити HTML більш структурованим і XML-сумісним. XHTML 1.0 було прийнято у 2000 році.</p>

        

        <h3>HTML5 (2014)</h3>

        <p>HTML5 є сучасною версією HTML, яка включає нові елементи для мультимедіа, графіки, інтерактивних додатків та поліпшення семантичної структури документів. HTML5 став офіційним стандартом у 2014 році і постійно оновлюється для підтримки нових технологій.</p>
